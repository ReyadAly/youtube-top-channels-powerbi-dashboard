# youtube-top-channels-powerbi-dashboard
Power BI dashboard analyzing the most subscribed YouTube channels: subscribers, categories, brand presence, and geographic distribution.

**Date:** 11th March, 2026  
**Tool:** Power BI (Power Query)  
**Author:** Reyad Aly  
**Dataset:** [Most Subscribed YouTube Channels — Kaggle](https://www.kaggle.com/datasets/sukhmandeepsinghbrar/most-subscribed-youtube-channel)


## Dashboard Highlights
- Channels with vs without brand presence
- Top 10 channels by subscribers
- Total subscribers by content category
- Channel distribution by country
- Total subscribers overall

## Data Cleaning Steps (Power Query)
- Promoted headers and changed data types
- Removed duplicates and blank rows
- Cleaned and trimmed text columns
- Split column by delimiter for language/country separation
- Removed irrelevant columns
- Added custom column for additional categorization
- Reordered and renamed columns for clarity
- Applied multiple type changes for numeric fields
